0 ไม่รับทราบข้อมูล ไม่มีการปฏิบัติ
1 รับรู้ถึงข้อกำหนด สามารถให้รายละเอียดการปฏิบัติได้ ให้ข้อมูลว่าเคยปฏิบัติแต่ขาดเอกสารยืนยัน หรือเพื่อตรวจสอบย้อนหลัง
2 มีการปฏิบัติที่เห็นได้ชัดเจน มีรูปแบบชัดเจน แต่ขาดเอกสารเพื่อยืนยัน หรือเพื่อตรวจสอบย้อนหลัง
3 มีการปฏิบัติที่เห็นได้ชัดเจน มีรูปแบบชัดเจน มีเอกสารเพื่อยืนยัน และตรวจสอบย้อนหลังได้
4 มีการปฏิบัติที่เห็นได้ชัดเจน มีรูปแบบชัดเจน มีเอกสารเพื่อยืนยัน และตรวจสอบย้อนหลังได้ และมีแผนหรือแนวทางในการพัฒนา
หมายเหตุ
1. Class A หมายถึง Multiplace HBC
2. Class B หมายถึง Monoplace HBC
3. หัวข้อที่มีเครื่องหมาย * (เช่น HBOPC 1.2.3*) หมายถึง MUST (ข้อบังคับ)
4. หมายเลขของหัวข้อจะไม่เรียงลำดับ บางหัวข้อจะข้ามไปเนื่องจากยกเลิก
5. ลงคะแนนให้ครบทุกหัวข้อ
